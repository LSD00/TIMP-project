\doxysubsection{Подробное описание}
Класс окна регистрации новых пользователей. 


\begin{DoxyItemize}
\item Отвечает за сбор имени, электронной почты и пароля от нового пользователя и отправку этих данных на сервер (клиентскую логику) для регистрации. 
\end{DoxyItemize}
