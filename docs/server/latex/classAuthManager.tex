\doxysubsection{Подробное описание}
Главный класс-\/синглтон для управления процессом авторизации, регистрации и восстановления паролей. 


\begin{DoxyItemize}
\item Класс инкапсулирует работу с базой данных Postgre\+SQL (сохранение и проверка хешей паролей), отправку писем с кодами подтверждения через внешний API, а также управление временем жизни сессий верификации. 
\end{DoxyItemize}
