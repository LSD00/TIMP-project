Подробное описание

Главный класс-синглтон для управления процессом авторизации, регистрации и восстановления паролей.

  • Класс инкапсулирует работу с базой данных PostgreSQL (сохранение и проверка хешей паролей), отправку писем с кодами подтверждения через внешний API, а также управление временем жизни сессий верификации.
