\doxysubsection{Подробное описание}
Класс окна визуализации данных. 


\begin{DoxyItemize}
\item Отвечает за отрисовку графика функции на основе входных данных от пользователя (коэффициенты a, b, c), которые задаются при помощи ползунков. Для отрисовки используется сторонний компонент (например, QCustom\+Plot, внедренный в ui-\/\texorpdfstring{$>$}{>}widget). 
\end{DoxyItemize}
