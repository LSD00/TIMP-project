Вычисление значений кусочно-заданной функции для построения графика.

  • Функция генерирует массив значений Y на основе заданных коэффициентов a, b и c в диапазоне X от -10 до 10 с шагом 0.067.
  • Аргументы
    aКоэффициент для X < 0.
    bКоэффициент для 0 <= X < 1.
    cКоэффициент для X > 1.
    Возвращает
    QVector<double> Вектор вычисленных значений Y для построения графика.
