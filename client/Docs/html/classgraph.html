Подробное описание

Класс окна визуализации данных.

  • Отвечает за отрисовку графика функции на основе входных данных от пользователя (коэффициенты a, b, c), которые задаются при помощи ползунков. Для отрисовки используется сторонний компонент (например, QCustomPlot, внедренный в ui->widget).
