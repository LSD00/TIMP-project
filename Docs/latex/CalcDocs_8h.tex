Главная функция вычисления значений кусочно-\/заданной функции. 

Функция проходит по отрезку от x = -\/10.\+0 до x = 10.\+0 с шагом 0.\+5. Расчет значения Y зависит от положения X и пользовательских параметров a, b, c.

Главная функция вычисления значений кусочно-\/заданной функции.

Выполняется цикл из 41 шага (от -\/10 до +10). Логика математического расчета Y разделена на 3 ветви\+:
\begin{DoxyItemize}
\item При $ x < 0 $\+: $ y = \frac{a}{x^2 - 1} $. Реализована защита\+: если знаменатель близок к нулю, Y = 0.
\item При $ 0 < x < 1 $\+: $ y = \frac{1}{x} + b $.
\item При $ x > 1 $\+: $ y = \frac{c \cdot x}{x^2 + 1} $.
\item Если X не попадает ни в одно условие (например, ровно 0 или 1), Y остается равен 0.\+0.
\end{DoxyItemize}
