\doxysubsection{Подробное описание}
Главный класс TCP-\/сервера. 

Сервер принимает подключения от клиентов, читает JSON-\/сообщения, обрабатывает открытые запросы (авторизация, регистрация) и защищенные запросы (вычисление графиков), доступные только после входа.
