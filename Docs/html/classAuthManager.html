Подробное описание

Менеджер аутентификации и управления пользователями.

Класс реализует паттерн Singleton. Обеспечивает безопасное потоконезависимое взаимодействие с базой данных (PostgreSQL) для регистрации, входа и сброса пароля. Наследуется от QObject для использования таймеров и сигналов/слотов.
